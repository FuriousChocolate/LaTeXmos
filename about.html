About

LaTeXmos is a tool that allows you to create Desmos-pastable LaTeX out of LaTeX that is not normally possible to create in Desmos. It is not meant to create expressions that are functional. "So what's the point?" LaTeXmos is for textual purpouses only. If there is math in a graph that you would like to explain, you may use LaTeXmos to explain it more neatly inline. You may also use it in dynamic labels to create more advanced labels in your Desmos graphs.
